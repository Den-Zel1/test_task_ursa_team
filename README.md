# Esports Match Tracker

Легкий и быстрый генератор сайтов с расписанием киберспортивных матчей. Данные подтягиваются из PandaScore API.

##  Особенности
- **Автогенерация HTML**: Чистый CSS/HTML без тяжелых фреймворков.
- **SEO Ready**: Уникальные мета-теги для каждой страницы и микроразметка **Schema.org** (JSON-LD).
- **Защита лимитов**: Кэширование запросов (TTL 60с) и Rate Limiting (20 запр/мин).
- **Clean URL**: Доступ по прямым адресам `/yesterday`, `/today`, `/tomorrow`.
- **Reliability**: Multi-stage Docker сборка с автоматическим прогоном тестов.

##  Технологии
- **Python 3.13** + **FastAPI**
- **uv** (пакетный менеджер)
- **httpx** (асинхронные запросы)
- **Pytest** + **Respx** (тестирование)
- **Docker** + **Docker Compose**
